## 9. Intellectual Property Treatment

The following are distinct and not equivalent:

| Concept                | Meaning                                 |
| ---------------------- | --------------------------------------- |
| Source-code possession | Physical/technical possession of code.  |
| Source-code license    | A right to use code under stated terms. |
| Copyright ownership    | Ownership of the copyright in the code. |
| IP assignment          | Transfer of specified IP rights.        |

Any assignment must explicitly identify the rights and specifically identified
deliverables being assigned. Possession or a license does not, by itself, transfer
copyright or ownership. [LEGAL REVIEW REQUIRED]
