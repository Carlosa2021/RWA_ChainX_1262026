## 6. CONFLICTOS ENTRE SOCIOS

### 6.1. Deadlock (Bloqueo):

**Si hay desacuerdo irreconciliable:**

**Opción A: Russian Roulette**
```
1. Socio A ofrece comprar participación de Socio B a precio X
2. Socio B puede:
   - Vender a ese precio, o
   - Comprar participación de Socio A al mismo precio
```

**Opción B: Shotgun Clause**
```
1. Socio A hace oferta de compra
2. Socio B tiene 60 días para:
   - Aceptar y vender, o
   - Contrarrestar y comprar él
```

**Opción C: Mediación/Arbitraje**
```
1. Mediación (30 días)
2. Si no hay acuerdo: Arbitraje vinculante
3. Árbitro decide compra forzosa o liquidación
```

### 6.2. Protección Especial para Carlos:

**En caso de deadlock:**
- Carlos Bernal puede cancelar licencia del Software
- ChainX SL tiene 180 días para:
  - Comprar el Software (valoración independiente)
  - Desarrollar alternativa
  - Cerrar el negocio
- Carlos Bernal NO puede ser forzado a vender el Software
